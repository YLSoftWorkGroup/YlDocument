# 指令

# v-permissionSetting

功能权限控制通过v-permissionSetting来控制功能按钮是否渲染

# 用法
 <el-button 
    type="primary" 
    icon="plus" 
    size="small" 
    v-permissionSetting"Roles.Create"  
    @click="_add" >添加</el-button>

注意：如果按钮不需要权限控制，传入v-permissionSetting="show"即可。
